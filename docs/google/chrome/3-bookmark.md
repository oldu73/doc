## Restore from Bookmarks.bak (last session only)

⚠️ Works only if Chrome has NOT been restarted since the accident.

Steps

Close Chrome completely

Go to:

`C:\Users\<username>\AppData\Local\Google\Chrome\User Data\Default\`

Delete Bookmarks

Rename Bookmarks.bak → Bookmarks

Restart Chrome

Important notes

Bookmarks is a single JSON file (no extension)

Chrome reads it on startup

Bookmarks.bak is overwritten at every Chrome restart
