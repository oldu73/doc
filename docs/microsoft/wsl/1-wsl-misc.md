## Explorer

[SRC](https://www.howtogeek.com/426749/how-to-access-your-linux-wsl-files-in-windows-10/)

Windows explorer to retrieve files from WSL (terminal), type in:

```console
explorer.exe .
```

From Windows explorer address bar (path), type in:

```text
\\wsl$
```
