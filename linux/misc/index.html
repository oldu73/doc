median.awk

#/usr/bin/env awk
{
    count[NR] = $1;
}
END {
    if (NR % 2) {
        print count[(NR + 1) / 2];
    } else {
        print (count[(NR / 2)] + count[(NR / 2) + 1]) / 2.0;
    }
}

$ cat test.txt | awk -f median.awk
11

Check equal number of values below and above median

$ cat test.txt | awk '{if($1 < 11) print $1}' | wc -l
3
$ cat test.txt | awk '{if($1 > 11) print $1}' | wc -l
3
